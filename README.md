## ⚙️ To Run the Project
1. **First, pull the repo to yourself:**
```bash
git clone https://github.com/utkbkts/product-management-system.git
```
2.**Go to the project directory and install the necessary dependencies:**
 ```bash
cd frontend
npm install
```
3.**Run the project:**
 ```bash
npm run dev
```
4.**Compile locally:**
```bash
npm run build
```
5.**.env-Backend**
```bash
DATABASE_URL=""

AUTH_SECRET=""

UPLOADTHING_SECRET=''
UPLOADTHING_APP_ID=''
UPLOADTHING_TOKEN=''

RESEND_API_KEY=""
```
